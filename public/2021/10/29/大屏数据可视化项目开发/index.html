大屏数据可视化项目开发

1. 创建项目
vue create datav

2. 安装依赖

npm install echarts
3. 页面拆分

（1）热销商品（基于饼图封装）
（2）客户分布（基于地图封装）
（3）销售数据（基于柱图封装）
（4）库存分析（基于饼图封装）
（5）销售走势（基于折线图封装）
（6）商家统计（基于柱图封装）

4. 热销商品组件开发
<template>
    <div class="c-container">
        <div ref="hot_ref"></div>
    </div>
</template>

<script>
export default {
    data () {
        return {
            initOption: {
                title: {
                    text: '热销商品销售统计',
                    top: 20,
                    left: 20
                },
                legend: {
                    top: '20%',
                    icon: 'circle'
                },
                tooltip: {
                    show: true
                },
                series: [
                    {
                        type: 'pie',
                        label: {
                            show: false
                        }
                    }
                ]
            },
            timer,
            chartInstance: null,
            currentIndex: 0
        }
    },
    methods: {
        // 初始化
        initChart () {
            this.chartInstance = this.$echarts.init(this.$refs.hot_ref, 'chalk')
        },
        // 动态展示数据
        updateChart () {
            this.chartInstance.setOption(updateOption)
        },
        // 自适应窗口改变
        updateScreen () {
            const titleFontSize = this.$refs.hot_ref.offsetWidth / 100 * 3.6
            const screenOption = {
                title: {
                    textStyle: {
                        textStyle: {
                            fontSize: titleFontSize / 1.5
                        }
                    }
                }
            }
        },
        startInterval () {
            if (this.timer) {
                clearInterval(this.timer)
            }
            this.timer = setInterval(() => {
                this.currentIndex++
                if (this.currentIndex > this.allData.length - 1) {
                    this.currentIndex = 0
                }
                this.updateChart()
            }, 2000)
        }
    }
}
</script>
